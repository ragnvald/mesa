Welcome to the MESA tool. The tool is based on a method developed by UNEP-WCMC and the Norwegian Environment Agency. You are now in the quick guide for our tool. Our aim with this page is to give you some basic information about how to use the system.

Why this system? We built MESA to streamline and enhance sensitivity analysis by applying the MESA method, which is typically complex and prone to errors when done manually in GIS tools. The software allows users to perform this analysis faster and with greater accuracy, reducing the likelihood of mistakes.
